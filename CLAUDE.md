# CLAUDE.md

このファイルは、Claude Code (claude.ai/code) がこのリポジトリのコードを扱う際のガイダンスを提供します。

## プロジェクト概要

これはMarp（Markdown Presentation Ecosystem）を使用してプレゼンテーションスライドを作成する日本語のスライドリポジトリです。Markdownで作成されたプレゼンテーション資料が含まれています。

## アーキテクチャ

- **スライド**: ルートディレクトリのMarkdownファイルにMarpフロントマターを含むスライドコンテンツ
- **スタイル**: `styles/`ディレクトリのカスタムCSSテーマ、主に「lt」テーマを使用
- **画像**: プレゼンテーションで使用する`images/`ディレクトリの画像アセット
- **出力**: MarkdownスライドからPDFファイルを生成

## 主要技術

- **Marp**: Markdownベースのプレゼンテーションツール
- **カスタムCSSテーマ**: カバー、セクション、コンテンツ用の専用スタイリングを持つ「lt」テーマ

## ファイル構造

- 各イベント専用ディレクトリ（例：`20250827_AI コーディングAgent 活用LT会/`）: 日付とイベント名でディレクトリを整理し、スライドファイル（`.md`）とPDFファイル（`.pdf`）、使用する画像を含む
- `styles/lt.css`: 日本語プレゼンテーション用スタイリングのカスタムMarpテーマ
- `images/`: 共通で使用するプレゼンテーション画像とアセット
- `cheatsheet/cheatsheet.md`: Marpの使用パターンを示すテンプレート/サンプルスライド
- `old/`: 古いディレクトリ構造で作成されたファイルのアーカイブ

## 開発ワークフロー

このプロジェクトは開発にMarp for VS Code拡張機能が必要です。スライドはMarp固有のフロントマターを含むMarkdownで記述され、プレゼンテーションに変換されます。

## Marp設定パターン

スライドは一貫したフロントマターを使用:
```yaml
---
marp: true
lang: ja
theme: lt
size: 16:9
math: katex
title: "スライドタイトル"
paginate: true
---
```

## CSSクラス

カスタム「lt」テーマは以下の専用スライドクラスを提供:
- `cover`: 背景画像付きタイトルスライド
- `toc`: 目次スタイリング
- `full_screen`: 強調用の中央揃えコンテンツ
- `section_title`: 背景付きセクション区切り
- `appendix`: 付録スライドスタイリング
- `back_cover`: 終了スライド

## コンテンツ言語

すべてのコンテンツは日本語です。スライドを編集または新規作成する際は日本語を維持してください。
